# FORGE Verify Command

Standalone verification command using the verification-before-completion skill.

## Usage

```bash
/forge:verify "All tests pass"
/forge:verify "No console errors"
/forge:verify "Performance benchmarks met"
```

## Purpose

Provides evidence-based validation outside the main workflow.

## Process

1. Define verification criteria
2. Gather evidence
3. Validate against criteria
4. Report pass/fail with evidence
